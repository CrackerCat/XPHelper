### 包含的功能

- Field Method Constructor的快速查找与调用(带缓存)
- 启动未注册在manifest清单的Activity
- 往宿主APP注入模块的Res资源
- 快速使用Dexkit来查找方法 并自动缓存方法与回收内存
- 附带FastKV(Java编写,兼容性更好,传闻比MMKV效率更高)存储框架的封装类
